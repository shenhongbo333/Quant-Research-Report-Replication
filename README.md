# Quant-Research-Report-Replication
Contains all kinds of report replications created and updated by a quant student.
## 山西证券研报复现folder
主要复现山西证券研报《因子测试框架和估值因子初测》中的估值因子测试思路。项目以 A 股市场为研究对象，选取 BP 因子，即账面市值比因子，作为核心测试因子，完成从股票池构建、因子清洗、标准化、中性化处理，到因子有效性检验、分层回测和绩效评估的完整流程。
## 中信期货_hurst folder
主要复现并扩展基于 Hurst 指数的商品期货 CTA 日频趋势策略。项目完成了从主力连续合约数据获取、信号生成、参数搜索、组合构建到样本内/样本外回测的完整研究流程。策略支持全局固定参数、单品种最优参数和多参数 ensemble 等不同实验模式，用于比较参数稳定性并降低过拟合风险。仓库中同时整理了回测结果、参数敏感性分析和样本外表现评估，方便后续继续迭代和复现实验。
